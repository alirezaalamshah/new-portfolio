Dieses Projekt lief ursprünglich auf WordPress und war an seine Grenzen gestoßen. Ich habe die gesamte Seite mit React im Frontend und Django im Backend neu entwickelt — für bessere Leistung, höhere Skalierbarkeit und eine modernere Architektur.

## Was sich geändert hat

- Vollständige Trennung von Frontend und Backend über eine API
- Völlige Designfreiheit ohne die Einschränkungen von WordPress-Themes
- Eine Architektur, die neue Funktionen einfach erweiterbar macht
