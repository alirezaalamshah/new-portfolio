This project originally ran on WordPress and had outgrown its limits. I rewrote the entire site with React on the front end and Django on the back end for better performance, higher scalability, and a more modern architecture.

## What changed

- Full separation of front end and back end through an API
- Complete design freedom, without WordPress theme constraints
- An architecture that makes adding new features straightforward
