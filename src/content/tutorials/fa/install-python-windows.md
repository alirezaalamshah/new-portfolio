اگر می‌خواهید برنامه‌نویسی را با پایتون شروع کنید، اولین قدم نصب درست آن است — و بیشترِ مشکلاتی که مبتدی‌ها روز اول با آن‌ها روبه‌رو می‌شوند (مثل خطای «python is not recognized») دقیقاً از همین‌جا می‌آید. در این آموزش قدم‌به‌قدم پایتون را نصب می‌کنیم و اولین برنامه را اجرا می‌کنیم.

## قدم اول: دانلود نسخه درست

به سایت رسمی [python.org/downloads](https://www.python.org/downloads/) بروید و روی دکمه زرد **Download Python** کلیک کنید. همیشه از سایت رسمی دانلود کنید، نه سایت‌های واسطه — هم امن‌تر است و هم همیشه آخرین نسخه را می‌گیرید.

## قدم دوم: مهم‌ترین تیک عمرتان را بزنید

نصب‌کننده را که باز کردید، **قبل از کلیک روی Install Now** پایین پنجره را نگاه کنید:

- تیک **Add python.exe to PATH** را حتماً بزنید ✅

این تیک به ویندوز می‌گوید دستور `python` را در هر پوشه‌ای بشناسد. اگر فراموشش کنید، به محض باز کردن ترمینال با این خطا روبه‌رو می‌شوید:

```text
'python' is not recognized as an internal or external command
```

اگر قبلاً بدون این تیک نصب کرده‌اید، ساده‌ترین راه این است که نصب‌کننده را دوباره اجرا کنید، **Modify** را بزنید و در بخش Advanced Options گزینه «Add Python to environment variables» را فعال کنید.

## قدم سوم: تست نصب

ترمینال را باز کنید (کلید Windows را بزنید، تایپ کنید `cmd` و Enter) و بنویسید:

```bash
python --version
```

اگر چیزی شبیه این دیدید، نصب موفق بوده:

```text
Python 3.13.1
```

`pip` (نصب‌کننده پکیج‌های پایتون) هم خودکار نصب شده؛ با این دستور مطمئن شوید:

```bash
pip --version
```

## قدم چهارم: اولین برنامه

یک پوشه برای تمرین‌هایتان بسازید، داخلش فایلی به اسم `hello.py` ایجاد کنید و این را بنویسید:

```python
name = input("اسمت چیه؟ ")
print(f"سلام {name}! به دنیای پایتون خوش اومدی 🐍")
```

حالا در ترمینال، داخل همان پوشه:

```bash
python hello.py
```

برنامه اسمتان را می‌پرسد و سلام می‌کند — تبریک، شما الان برنامه‌نویس پایتون هستید!

## نکته حرفه‌ای: از همین اول با محیط مجازی کار کنید

وقتی جلوتر بروید و پکیج نصب کنید، خیلی زود به تداخل نسخه‌ها می‌خورید. عادت طلایی این است که برای هر پروژه یک محیط مجازی بسازید — روشش را در [راهنمای کامل محیط مجازی در پایتون](/tutorials/python-virtualenv-guide/) نوشته‌ام.

## تمرین

برنامه‌ای بنویسید که سال تولد را بپرسد و سن را حساب کند. راهنمایی: `input` همیشه متن برمی‌گرداند؛ با `int()` به عدد تبدیلش کنید.
